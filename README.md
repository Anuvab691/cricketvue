# CricketVue | Premium Exchange Terminal

The ultimate cricket prediction and insights platform, powered by professional real-time data from the **Sportbex Network**.

## Getting Started

1. **Authentication**: Sign up or Log in using your email and password.
2. **Dashboard**: View live and upcoming matches synced automatically from the Sportbex API.
3. **Live Insights**: Get AI-powered match predictions and commentator facts.
4. **Mock Betting**: Use virtual tokens to place stakes on match winners and fancy markets.

## Data Synchronization
The system is configured for **Live Network Sync**. It fetches the latest data from Sportbex using a secure API key.

## Technical Stack
- **Framework**: Next.js 15 (App Router)
- **Database**: Firebase Firestore
- **Auth**: Firebase Authentication
- **AI**: Google Genkit (Gemini 1.5 Flash)
- **UI**: Tailwind CSS & ShadCN UI
- **Primary Data Provider**: Sportbex API (/live-score/match/live)
